# FullHousePotential

Webapplication for tracking home poker games. 
Make a group and add your friends, from there you can always add new players and nights (night being a single game played).
The application tracks statistics over multiple nights, produces a leaderboard so you can see what is the stand of your totals.

## Usage

Navigate to the [GitHub pages](https://olivertenhoor.github.io/FullHousePotential/index.html), from there everything should be clear on how to use it.
